The Panora API is organized around [REST](https://en.wikipedia.org/wiki/REST).
Our unified API has predictable resource-oriented URLs, accepts form-encoded request bodies, returns JSON-encoded responses, and uses standard HTTP response codes, authentication, and verbs.

## Base URL

<Tabs>
  <Tab title="Cloud">
      ```https://api.panora.dev/```
  </Tab>

  <Tab title="Self-Hosted">
      ```http://localhost:3000/```
  </Tab>
</Tabs>
